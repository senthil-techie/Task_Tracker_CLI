A simple Command Line Interface (CLI) based Task Tracker application built using Java. This project demonstrates core Object-Oriented Programming (OOP) principles, CRUD operations, and collection handling using ArrayList.

Add new tasks

Update existing tasks

Delete tasks

View all tasks

Track task status (Pending / In Progress / Completed)

Clean console-based output

Java

OOP Concepts (Encapsulation, Abstraction)

ArrayList (Java Collections Framework)

Task Class

Stores task details (id, description, status)

Contains getters, setters, and toString()

TaskManager Class

Maintains a list of tasks

Handles add, update, delete, and display operations

Main Class

Entry point of the program

Calls TaskManager methods
